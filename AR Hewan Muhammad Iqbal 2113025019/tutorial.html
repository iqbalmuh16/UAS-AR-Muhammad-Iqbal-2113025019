Tutorial Penggunaan AR

Selamat datang di tutorial penggunaan aplikasi AR untuk edukasi hewan. Berikut adalah langkah-langkah untuk memulai:

Langkah 1: Persiapkan Marker

Pastikan Anda memiliki marker AR yang disediakan. Anda dapat melihat gambar marker di halaman utama aplikasi.

Langkah 2: Mulai Aplikasi AR

Klik tombol "Start" pada halaman utama untuk memulai aplikasi AR.

Langkah 3: Posisikan Marker

Pindai marker dengan kamera ponsel Anda. Pastikan marker berada dalam pandangan kamera untuk mengaktifkan model 3D hewan.

Langkah 4: Interaksi dengan Model 3D

Setelah model 3D muncul anda bisa melihat hewan yang di tampilkan
